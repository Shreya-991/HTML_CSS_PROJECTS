A modern chair product landing page built using HTML and CSS while learning frontend web development from an online course.

## Features
- Modern UI design
- Product showcase section
- Color selection options
- Responsive layout
- Smooth styling using CSS

## Technologies Used
- HTML5
- CSS3

## Learning Outcome
Through this project, I practiced:
- Flexbox layout
- CSS positioning
- Styling buttons and sections
- Working with images
- Building modern UI components

Completed as a practice project during learning.
