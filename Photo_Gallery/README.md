# Photo Gallery

A simple and responsive Photo Gallery project built using HTML and CSS while learning web development through an online course.

## 📌 Features

- Responsive photo gallery layout
- Clean and modern UI
- Image hover effects
- Organized project structure
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```bash
photo-gallery/
│── index.html
│── style.css
│── images/
```

## 📖 What I Learned

- Creating responsive layouts using CSS
- Using Flexbox/Grid for alignment
- Adding hover effects to images
- Structuring frontend projects properly
- Improving HTML and CSS fundamentals
